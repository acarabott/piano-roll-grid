Musical Grid Layers

Key Commands

Q Layer mode O
W Note mode P
0 - 9 Set layer subdivision
A S Cycle through layers K L
⌥ Alt Hold to copy
⌫ Backspace Delete selected note / layer
⇧ Shift Hold to disable snapping (on by default)
Space Play / Pause Playback

What?

This is a prototype for an alternative to having a fixed time grid on a piano roll. Instead you can lay down multiple grids with different subdivisions. New grids and new notes can be snapped to existing grids.

Think of it like a piece of paper, and the grids as stencils that you move about to assist your drawing.

Instructions

There are two modes

  1. Layers - Create rhythm templates for drawing notes
  2. Notes - Create musical notes

Switch between modes using Q (Layers) and W (Notes), or the drop down menu.

Click and drag to create new layers/notes. They will automatically snap to existing layers. Hold ⇧ Shift to disable snapping.

Set the subdivision of the focused layer by typing with the number keys. It can be any value, after half a second without typing, the value will be set.

Use A and S to cycle through the stack of layers (this includes the background layer).
The focused layer will be highlighted in blue.

Move the focused layer by grabbing one of its edges, the cursor will be (TODO CURSOR).
Move notes by grabbing anywhere inside of the note.
Copy by holding ⌥ Alt.

Resize layers by grabbing one of their corners, the cursor will be (TODO CURSOR)
Resize notes by grabbing their left or right edges, cursor (TODO CURSOR)

Delete the focused layer or note with ⌫ Backspace

Why?

Having a single fixed grid (usually 16th notes) makes the default very inviting and alternatives less accessible. Without a global default, you can explore other rhythms.
It also makes it easier to do things that a global grid makes difficult, like nested tuplets or starting a tuplet on the off-beat.
New tools make for new approaches!
